# Public Assets - Placeholder Guide

This folder should contain your actual images. Below are the required files:

## Hero Images
- `hero-eyes.webp` - Main hero background (anime eyes close-up, 1920x1080)
- `about-image.webp` - About section image (800x800)
- `about-hero.webp` - About page hero (1200x800)

## Product Images (600x600 each)
Create a `products/` subfolder with:
- gojo.webp
- tanjiro.webp
- luffy.webp
- naruto.webp
- eren.webp
- zoro.webp
- sukuna.webp
- itachi.webp

## Collection Banners (800x600 each)
Create a `collections/` subfolder with:
- jjk.webp (Jujutsu Kaisen)
- ds.webp (Demon Slayer)
- op.webp (One Piece)

## Testimonial Avatars (100x100 each)
Create an `avatars/` subfolder with:
- avatar1.webp
- avatar2.webp
- avatar3.webp

## Quick Placeholder Solution

For development, you can use placeholder images:

1. Go to https://placehold.co/
2. Generate images like:
   - https://placehold.co/600x600/0A0F1F/00A8FF?text=Gojo
   - https://placehold.co/1920x1080/02060F/00A8FF?text=AKUSHO

Or download anime images from:
- Unsplash (free)
- Pinterest (reference only)
- Official anime sources (for personal projects)
